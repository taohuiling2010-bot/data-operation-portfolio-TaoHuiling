# 零售行业多维度销售分析与 Power BI 可视化项目
项目周期：2026.02 - 2026.03
项目角色：独立负责人
核心技术栈：SQL、Power BI（DAX / 数据建模）
项目背景：模拟企业零售业务数据运营实际场景，完成门店 / 大区 / 商品多维度销售分析，搭建可视化看板实现业务数据实时监控，为运营决策提供数据支撑，复刻企业数据运营核心工作流程。

## 核心工作：
通过 SQL 完成零售业务数据的多维度取数与聚合分析，输出标准化分析数据；
在 Power BI 中完成数据清洗，构建大区 - 门店 - 商品三层数据模型，编写 DAX 度量值实现核心指标动态计算；
搭建 3 大交互式可视化看板，实现 KPI 视觉预警、销售极值识别，打造宏观到微观的分层分析体系；
分析业务数据差异，提出备货、清库、陈列等可执行运营建议，形成数据到业务的落地闭环。

## 核心成果：
打造可复用的零售销售分析模板，实现 6 大核心业务指标的实时监控；
落地 “数据采集 - 分析 - 可视化 - 业务落地” 的完整数据运营闭环，体现数据驱动业务优化的核心能力；
掌握企业级数据建模与 Power BI 交互式看板搭建的实操方法，贴合数据运营岗位实际工作需求。
