# 豆瓣电影 Top250 数据分析项目
1.	项目周期：2026.02 - 2026.03
2.	项目角色：独立负责人
3.	核心技术栈：Python（requests/etree/Pandas/NumPy/Matplotlib）、Excel
4.	项目背景：通过网络数据爬取与多维度数据分析，挖掘豆瓣电影 Top250 的核心特征与分布规律，实操非结构化数据处理、数据清洗、统计分析与可视化的全流程，夯实数据分析基础能力。

## 核心工作：
1.	使用 Python 爬虫获取豆瓣电影 Top250 的 2238 条核心数据，实现非结构化数据的结构化采集与格式化存储；
2.	用 Pandas 完成数据清洗，包括缺失值处理、异常值剔除、字段标准化，保障数据质量；
3.	对电影类型、年份、国家等维度做独热编码与统计分析，挖掘数据分布特征；
4.	通过 Matplotlib 制作多维度可视化图表，计算指标间相关系数，挖掘数据关联规律。

## 核心成果：
1.	掌握 Python 网络爬虫与非结构化数据处理的实操方法，实现数据的精准采集与格式化；
2.	熟练运用 Pandas、Matplotlib 完成数据清洗、统计分析与可视化，夯实数据分析核心能力；
3.	学会通过统计分析挖掘数据间的关联规律，形成从数据采集到规律分析的完整实操闭环。
